# API de Pedidos

API REST desenvolvida em Node.js para gerenciamento de pedidos.

## Tecnologias utilizadas

- Node.js
- Express
- Git
- GitHub

## Estrutura do projeto

src/
controllers/
routes/
services/
utils/

## Endpoints

Criar pedido

POST /order

Buscar pedido

GET /order/:id

Atualizar pedido

PUT /order/:id

Excluir pedido

DELETE /order/:id

## Como executar o projeto

Instalar dependências

npm install

Rodar servidor

node index.js

Servidor roda em:

http://localhost:3000
